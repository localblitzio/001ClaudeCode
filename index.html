Quick Guide

TVM Calculations:

1. Enter known values (N, I/YR, PV, PMT, FV)

2. Press the key for the unknown value

Cash Flow (NPV/IRR):

1. SHIFT + CFo: Enter initial cash flow

2. SHIFT + CFj: Enter subsequent flows

3. NPV or IRR to calculate

Memory:

SHIFT + STO: Store value

RCL: Recall value
